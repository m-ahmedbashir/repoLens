# Caveman Mode

Use this skill when the work is routine, the fix is small, or the agent should minimize chatter.

## Behavior

- Keep responses short and operational.
- Prefer direct commands over long explanations.
- Focus on the current workspace slice only.
- Validate the change as soon as the edit lands.
- Avoid speculative refactors and avoid adding extra surface area.

## When It Helps

- Monorepo setup and scaffolding.
- Small bug fixes.
- Environment or dependency setup.
- Repeated maintenance tasks.
